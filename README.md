# HTML & CSS Fundamentals Project

This repository contains practice files and examples covering the core concepts of **HTML and CSS used for building modern websites**. The project demonstrates how basic web technologies work together to structure and style webpages.

---

## 📚 Topics Covered

### HTML Fundamentals

* Hyperlinks
* Images
* Audio
* Video
* Favicons
* Text Formatting
* `span` and `div`
* Lists
* Tables
* Buttons
* Forms
* Headers and Footers

---

### CSS Fundamentals

* Introduction to CSS
* Colors
* Fonts
* Borders
* Shadows
* Margins
* Float
* Overflow
* Display Property
* Height and Width
* Positioning
* Background Images

---

### Advanced CSS Concepts

* CSS Combinators
* Pseudo-Classes
* Pseudo-Elements
* Pagination
* Dropdown Menus
* Navigation Bar
* Website Layout Design
* Image Gallery
* Icons

---

### Modern Layout & Effects

* Flexbox
* Transformations
* Animations

---

## 🎯 Purpose of this Repository

The purpose of this project is to:

* Learn the **core building blocks of web development**
* Understand how **HTML structures content**
* Learn how **CSS styles and layouts webpages**
* Practice creating **responsive and visually structured web pages**
