## How to Use

1. Start with **01-introduction-to-html** (or any topic you want to learn)
2. Open the `index.html` file in each folder
3. Use Live Server extension to see changes in real-time
4. Modify code, experiment, and learn!
5. Each folder is independent - you can practice at your own pace

## Tips

- 📁 Each folder contains `index.html` and `style.css` for the topic
- 🔗 Test hyperlinks between topics by creating links in your index.html files
- 💾 Save your work frequently
- 📚 Refer back to previous topics when needed
- 🎯 Try combining multiple topics in larger projects

## Getting Started

1. Open a topic folder
2. Start coding in `index.html`
3. Apply CSS in `style.css`
4. Use Live Server to preview changes

Happy Learning! 🚀
