Lists 📄

Lists organize information in a structured way. There are 2 main types:

Unordered List (Bullet Points)

  • First item - gets a bullet point
  • Second item - also gets a bullet point
  • Third item - bullet point too!

Ordered List (Numbers)

  1. First step - numbered 1
  2. Second step - numbered 2
  3. Third step - numbered 3
