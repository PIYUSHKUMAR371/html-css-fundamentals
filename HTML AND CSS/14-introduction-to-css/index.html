Introduction to CSS 🎨

CSS (Cascading Style Sheets) is used to style HTML elements.

There are 3 ways to apply CSS:

  1. Inline CSS - type="style" directly on HTML tag (NOT recommended - messy)
  2. Internal CSS - <style> tags inside <head> (OK, but harder to reuse)
  3. External CSS - separate .css file (BEST! Easy to reuse and maintain)
